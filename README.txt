Overview
The Modern Weather App displays current weather, hourly forecasts, and daily summaries. It uses clean, modern UI styles with CSS variables for easy customization.

Features
- City-based search
- Current weather: temperature, feels like, conditions, humidity, wind speed, pressure
- Hourly & daily forecast cards
- Geolocation for user's current weather
- Loading animations and error handling
- Fully responsive with smooth transitions

Demo
A live demo will be shared once deployed on GitHub Pages.

Tech Stack
- HTML5, CSS3 (with CSS variables), Vanilla JS (ES6+)
- API: Open-Meteo (no API key) or optionally OpenWeatherMap

Getting Started
