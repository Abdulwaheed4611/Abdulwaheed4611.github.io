Overview
The Modern Weather App displays current weather, hourly forecasts, and daily summaries. It uses clean, modern UI styles with CSS variables for easy customization.

Features
- City-based search
- Current weather: temperature, feels like, conditions, humidity, wind speed, pressure
- Hourly & daily forecast cards
- Geolocation for user's current weather
- Loading animations and error handling
- Fully responsive with smooth transitions

Demo
A live demo will be shared once deployed on GitHub Pages.

Tech Stack
- HTML5, CSS3 (with CSS variables), Vanilla JS (ES6+)
- API: Open-Meteo (no API key) or optionally OpenWeatherMap

Getting Started
Prerequisites
- Modern browser (Chrome, Firefox, Safari, Edge)
- Optional: OpenWeatherMap API key if using that service

Installation
- Clone the repository:
  git clone https://github.com/your-username/weather-app.git
  cd weather-app
- Optional: install live server
  npm install -g live-server
  live-server

Usage
1. Open index.html in browser or via live-server.
2. Enter city in search bar and click "Get Weather" or press Enter.
3. Or click 📍 to get local weather.
4. See current, hourly and daily weather information.

You can customize color themes in styles.css via CSS variables.
